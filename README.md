Personal project involving C# and databases with DAPPER. 
NOTE: The project is still WIP, meaning the name conventions wont be final, neither will the debug messages stay in the code. Also, code snippets are to be credited yet (which will be both in this git page and in the code). 

The final server will be open sourced, anybody can download the project and open their server. However, it is advisable to make changes in order to preserve server identity (models, sounds, etc)
Translations for english, spanish and chinese will be provided in the final versions, and along the project updates.
